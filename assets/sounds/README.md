# Focus Music

## Required File

Please add a focus music file to this directory with the name:
- `focus_music.mp3` (recommended)

## File Requirements

- Format: MP3, WAV, or M4A
- Duration: 2-10 minutes (will loop automatically)
- Style: Lo-fi, ambient, white noise, or any relaxing background music
- Volume: Moderate level (will be adjustable in-app)

## Recommended Sources

- Free music libraries (YouTube Audio Library, Free Music Archive)
- Lo-fi music compilations
- Nature sounds or white noise
- Study/focus playlists

## Note

The app expects the file to be named `focus_music.mp3`. If you want to use a different name, update the constant in `lib/core/constants/app_constants.dart`.
